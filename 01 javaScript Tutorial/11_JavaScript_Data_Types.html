JavaScript Data Types

JavaScript Data Types
=======================

  var length = 16;                            // Number
  var lastName = "Johnson";                   // String
  var x = {firstName:"John", lastName:"Doe"}; // Object


JavaScript Types are Dynamic
==============================

JavaScript has dynamic types: the same variable can be used to hold
different data types:

> JavaScript Strings
> JavaScript Numbers
> JavaScript Booleans
> JavaScript Arrays
> JavaScript Objects
> The typeof Operator
> Undefined values and data types
      var car;          // Value is undefined, type is undefined
      car = undefined;  // Value is undefined, type is undefined

> Empty Values
    var car = "";       // The value is "", the typeof is "string"

> Null

> Difference Between Undefined and Null
    typeof undefined     // undefined
    typeof null          // object

    null === undefined   // false
    null == undefined    // true


Primitive Data
=================

A primitive data value is a single simple data value
with no dditional properties and methods.

The 'typeof' operator can return one of these primitive types:
  > string
  > number
  > boolean
  > undefined                                          -

 Complex Data
===============

The 'typeof' operator can return one of two complex types:
  > function
  > object
  > arrays

The 'typeof' operator returns "object" for
objects, arrays, and null.

The typeof operator does not return "object" for functions.
